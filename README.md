#  RegistrySystem

A desktop application built with **Python** and **PySide6** for managing population records using a **MySQL database**.

This system provides a simple and user-friendly interface to register, view, update, and manage citizen data efficiently.

---

##  Features

-  Add new population records  
-  Update existing records  
-  Delete records  
-  View all registered citizens  
-  Search and manage data easily  
-  Persistent storage using MySQL  
-  Modern desktop GUI built with PySide6  

---

##  Technologies Used

- Python 
- PySide6 (Qt GUI framework)   
- MySQL   
- MySQL Connector  

---

##  Project Purpose

This project was developed to practice and demonstrate:

- Desktop application development with PySide6  
- GUI design using Qt framework  
- CRUD operations with MySQL  
- Integration between frontend (UI) and backend (database)  
- Real-world data management system structure  

---

##  How to Run

### 1. Install dependencies
    pip install pyside6 mysql-connector-python

### 2. Set up MySQL database
### 3. Run the application
    py main.py
